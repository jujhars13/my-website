### Building
```bash
docker run --rm -v ${pwd}/site:/srv/jekyll -p 127.0.0.1:4000:4000 \
  jekyll/jekyll jekyll build
  
  docker run --rm --label=jekyll --volume=${pwd}/site:/srv/jekyll -it -p 127.0.0.1:4000:4000 jekyll/jekyll
```
